# Kozmetik Mağazası E-Ticaret Sitesi

PHP ve MySQL kullanılarak geliştirilmiş e-ticaret projesi.

## Özellikler
- Ürün listeleme
- Ürün detay sayfası
- Sepet sistemi
- Sipariş yönetimi
- Admin paneli
- AJAX arama sistemi

## Kurulum
1. Projeyi XAMPP htdocs klasörüne kopyalayın.
2. Veritabanını phpMyAdmin'e aktarın.
3. Apache ve MySQL'i başlatın.
4. localhost/kozmetik_magazasi adresinden çalıştırın.
